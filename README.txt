Write Experience Reports and insert the reports in pages or posts. With the Experience Reports Gutenberg plugin you have countless setting options and can place the reports anywhere you want.

== Description ==


== Features ==
 * Slider for posts
 * Custom slider settings and templates
 * Create gallery for posts
 * Custom gallery settings and templates
 * Show posts as news
 * countless output options

* 3rd Party
  Activation information will be transmitted to `https://www.wwdh.de`. No personal data will be sent or stored.

== Installation ==

Follow the steps below to install the plugin

1. Upload the folder `wp-experience-reports` to the directory `/wp-content/plugins/`.
1. Activate the plugin via the menu `Plugins` in WordPress.
